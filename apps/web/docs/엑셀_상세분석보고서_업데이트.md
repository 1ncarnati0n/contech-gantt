## 4. 수식 패턴 분석

### 4.1 시트간 참조 패턴

모든 수량(E열)은 `동,층별물량표` 시트의 특정 셀을 참조합니다:

- **직접 참조**: `='동,층별물량표'!D6` (버림 형틀)
- **비율 참조**: `='동,층별물량표'!F8*0.45` (옹벽철근 45%)
- **복합 참조**: `='동,층별물량표'!G9*0.6` (1차 타설 60%)

### 4.2 주요 수식 유형

#### 4.2.1 총작업인원 계산 (G열)
- **일수고정 방식**: `=ROUNDUP(E/F,0)` (수량 ÷ 일일 작업량)
- **장비기반 방식**: `=H*L` (1일 투입인원 × 직영 순작업일)

#### 4.2.2 1일 투입인원 계산 (H열)
- **일수고정 방식**: `=ROUNDUP(G/L,0)` (총작업인원 ÷ 직영 순작업일)
- **장비기반 방식**: `=K*5` 또는 `=K*6` (장비대수 × 인원수)

#### 4.2.3 장비대수 계산 (K열)
- **공식**: `=CEILING(MIN(2, E/I), 1)`
  - 최대 2대 제한
  - I열: 대당 타설량 기준값 (650 등)

#### 4.2.4 직영 순작업일 계산 (L열)
- **고정값**: 직접 입력 (1, 2, 5, 6, 17, 19, 22 등)
- **계산식**: `=MAX(1, IF(E/(F*H) - INT(E/(F*H)) < 0.5, ROUNDDOWN(E/(F*H),0), ROUNDUP(E/(F*H),0)))`
  - 소수점이 0.5 미만이면 내림, 이상이면 올림
  - 최소값 1일 보장

#### 4.2.5 총작업일수 계산 (N열)
- **공식**: `=ROUNDUP(L+M, 0)` (직영 순작업일 + 간접일)

### 4.3 수식 함수 사용 현황

- **ROUNDUP**: 93개 (가장 많이 사용)
- **ROUNDDOWN**: IF 조건문 내부에서 사용
- **CEILING**: 8개 (장비대수 계산)
- **MAX**: 직영 순작업일 계산에서 최소값 보장
- **MIN**: 장비대수 계산에서 최대값 제한
- **IF**: 소수점 처리 로직
- **INT**: 소수점 추출용
